# client/

Place client-provided photographer images in this folder.

Naming convention:
  client_[descriptor].webp

Examples:
  client_team_portrait_john.webp
  client_facility_exterior.webp
  client_job_hvac_building.webp

Preferred format: .webp, max 500 KB
If you receive .jpg or .png, convert to .webp before uploading.
